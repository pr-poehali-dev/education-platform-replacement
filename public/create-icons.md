# Инструкция по созданию иконок PWA

Для полноценной работы PWA необходимо создать две иконки:

1. **pwa-192x192.png** - иконка 192x192 пикселей
2. **pwa-512x512.png** - иконка 512x512 пикселей

## Сгенерированное изображение для иконки:
https://cdn.poehali.dev/projects/32d18763-e750-458c-bb3c-6faa05a6a29f/files/8b67369f-a009-4925-94e7-4d9d07265315.jpg

## Как создать иконки:

### Вариант 1: Онлайн-инструменты
1. Откройте https://realfavicongenerator.net/
2. Загрузите сгенерированное изображение
3. Скачайте готовые иконки нужных размеров
4. Переименуйте их в pwa-192x192.png и pwa-512x512.png
5. Поместите в папку `/public/`

### Вариант 2: Используйте существующий логотип
Если у вас уже есть логотип компании:
1. Откройте в любом редакторе (Photoshop, GIMP, Figma)
2. Измените размер до 192x192 и 512x512
3. Сохраните как PNG с прозрачным фоном
4. Поместите в папку `/public/`

### Вариант 3: Временные иконки
Можно использовать favicon.svg из проекта и конвертировать его в PNG нужных размеров.

## После добавления иконок:
Пересоберите проект - приложение готово к установке как PWA!
